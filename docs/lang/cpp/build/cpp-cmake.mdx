# Project tree

```shell
project
├── CMakeLists.txt
└── main.cpp
```

# CMakeLists.txt

```cmake title="CMakeLists.txt"
cmake_minimum_required(VERSION 3.15)

project(example LANGUAGES CXX)

add_executable(example main.cpp)
```
