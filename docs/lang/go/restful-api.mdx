## URI(Uniform Resource Identifiers)

`<schema>://<authority>/<path>[?<query>][#<fragment>]`

- `/`는 리소스 사이의 계층적 관계를 나타낼 때 사용한다.
- URI의 마지막에는 `/`를 붙이지 않는다.
- `-`를 사용할 수 있다.
- `_`를 사용하지 않는다.
- 대소문자를 구분한다. 일반적으로 소문자를 사용한다.
