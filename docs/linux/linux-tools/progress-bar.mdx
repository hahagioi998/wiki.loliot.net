## Copy

```shell
rsync -ah --info=progress2 <src> <dst>
```

## Compress

```shell
tar -czpf - <src> | (pv -p --timer --rate --bytes > <dst>.tar.gz)
```

```shell
pv <src>.tar.gz | tar -xzp
```
