## Installation

```shell
sudo apt install -y openssh-server openssh-client ssh-askpass ssh-askpass-gnome
```

## 키 생성

```shell
ssh-keygen -t ed25519 -C "your_email@example.com"
```

```shell
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

## .ssh/config

```shell
Host <alias>
  HostName <host>
  User <user>
  PreferredAuthentications publickey
  IdentityFile <path>
  AddKeysToAgent yes
```

## ssh agent 실행

```shell
eval "$(ssh-agent -s)"
```

아래 코드를 `.zshrc`에 추가하면 agent를 자동으로 실행합니다.

```shell
env=~/.ssh/agent.env

agent_load_env () { test -f "$env" && . "$env" >| /dev/null ; }

agent_start () {
    (umask 077; ssh-agent >| "$env")
    . "$env" >| /dev/null ; }

agent_load_env

# agent_run_state: 0=agent running w/ key; 1=agent w/o key; 2=agent not running
agent_run_state=$(ssh-add -l >| /dev/null 2>&1; echo $?)

if [ ! "$SSH_AUTH_SOCK" ] || [ $agent_run_state = 2 ]; then
    agent_start
    ssh-add
elif [ "$SSH_AUTH_SOCK" ] && [ $agent_run_state = 1 ]; then
    ssh-add
fi

unset env
```

## 키 등록

```shell
ssh-add ~/.ssh/<id file>
```

```shell
ssh-add -l
```

## 퍼블릭 키 접속할 곳에 등록

```shell
ssh-copy-id -i ~/.ssh/<id file>.pub <remote>
```

## SCP

source를 target으로 보낼 수 있는 명령어입니다.

원격지의 경우 **\<user\>@\<ip\>:\<path\>** 로 구성됩니다.

```shell
scp [options] <source> <target>
```

## 파일 권한

```shell
sudo chmod 700 ~/.ssh
sudo chmod 400 ~/.ssh/*
```
