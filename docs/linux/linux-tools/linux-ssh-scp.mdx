## 키 생성

```shell
ssh-keygen -t ed25519 -C "your_email@example.com"
```

```shell
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

## ssh agent 실행

```shell
eval "$(ssh-agent -s)"
```

## 키 등록

```shell
ssh-add ~/<ssh path>/<id file>
```

```shell
ssh-add -l
```

## 퍼블릭 키 접속할 곳에 등록

```shell
ssh-copy-id -i ~/<ssh path>/<id file>.pub <remote>
```

## SCP

source를 target으로 보낼 수 있는 명령어입니다.

원격지의 경우 **\<user\>@\<ip\>:\<path\>** 로 구성됩니다.

```shell
scp [options] <source> <target>
```

## 파일 권한

```shell
sudo chmod 700 ~/<ssh path>
sudo chmod 600 ~/<ssh path>/*
```
