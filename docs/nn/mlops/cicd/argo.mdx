## Installation

```shell
helm repo add argo https://argoproj.github.io/argo-helm \
&& helm repo update
```

```shell
mkdir -p cicd/argo/{events,workflows,cd}
```

```shell
helm search repo argo -l
```

### argo-events

```shell
helm show values argo/argo-events \
    --version 1.8.0 \
    > cicd/argo/events/values.yaml
```

```yaml title="cicd/argo/events/values.yaml"
# ...

namespace: cicd
singleNamespace: true
# ...
```

```shell
helm upgrade argo-events argo/argo-events \
    --install \
    --version 1.8.0 \
    -n cicd \
    --create-namespace \
    --values cicd/argo/events/values.yaml
```

### argo-workflows

```shell
helm show values argo/argo-workflows \
    --version 0.9.2 \
    > cicd/argo/workflows/values.yaml
```

```yaml title="cicd/argo/workflows/values.yaml"
# ...
controller:
  # ...

  # -- Specifies the container runtime interface to use (one of: `docker`, `kubelet`, `k8sapi`, `pns`, `emissary`)
  ## Ref: https://argoproj.github.io/argo-workflows/workflow-executors/
  containerRuntimeExecutor: emissary
```

```shell
helm upgrade argo-workflows argo/argo-workflows \
    --install \
    --version 0.9.2 \
    -n cicd \
    --values cicd/argo/workflows/values.yaml
```

### argo-cd

```shell
helm show values argo/argo-cd \
    --version 3.29.4 \
    > cicd/argo/cd/values.yaml
```

```yaml title="cicd/argo/cd/values.yaml"
# ...

server:
  # ...

  extraArgs:
    - --insecure # https://github.com/argoproj/argo-cd/issues/2953

  # ...

  # ConfigMap name: argocd-cm
  config:
    # Argo CD's externally facing base URL (optional). Required when configuring SSO
    url: <url>

    dex.config: |
      connectors:
        - type: oidc
          id: google
          name: Google
          config:
            issuer: https://accounts.google.com
            clientID: XXXXXXXXXXXXX.apps.googleusercontent.com
            clientSecret: XXXXXXXXXXXXX

  # ...

  # ConfigMap name: argocd-rbac-cm
  rbacConfig: {}
```

```shell
helm upgrade argo-cd argo/argo-cd \
    --install \
    --version 3.29.4 \
    -n cicd \
    --values cicd/argo/cd/values.yaml
```
