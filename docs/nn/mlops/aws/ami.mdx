## EC2 -> AMI

원하는 기본 이미지를 가지고 EC2를 생성합니다. Ubuntu 20.04 server의 경우 다 설치되어도 2 GB가 되지 않기 때문에, 8 GB 기본 볼륨만 마운트 된 상태에서 프로그램을 설치해도 문제 없습니다.

필요한 프로그램을 설치하고 해당 인스턴스를 선택한 후 `작업 -> 이미지 및 템플릿 -> 이미지 생성`을 누르고 저장합니다.

:::warning
기본 EBS 볼륨은 스냅샷으로 저장됩니다. 해당 AMI로 EC2를 만들 때, 연결된 스냅샷 볼륨 크기 이상만 가능합니다.
:::
