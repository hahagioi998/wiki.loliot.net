## metrics-server

```shell
helm repo add metrics-server https://kubernetes-sigs.github.io/metrics-server/
```

```shell
helm template metrics-server metrics-server/metrics-server > metrics-server.yaml
```

- `--kubelet-insecure-tls`
- `--v=2`

```shell
kubectl apply -n kube-system -f metrics-server.yaml
```

## Installation

```shell
wget https://raw.githubusercontent.com/kubernetes/dashboard/v2.4.0/aio/deploy/recommended.yaml -O k8s-dashboard.yaml
```

```shell
kubectl apply -f k8s-dashboard.yaml
```

## ServiceAccount

```yaml title="k8s-dashboard-admin.yaml"
apiVersion: v1
kind: ServiceAccount
metadata:
  name: k8s-dashboard-admin
  namespace: kubernetes-dashboard

---
apiVersion: rbac.authorization.k8s.io/v1
kind: ClusterRoleBinding
metadata:
  name: k8s-dashboard-admin
roleRef:
  apiGroup: rbac.authorization.k8s.io
  kind: ClusterRole
  name: cluster-admin
subjects:
  - kind: ServiceAccount
    name: k8s-dashboard-admin
    namespace: kubernetes-dashboard
```

```shell
kubectl apply -f k8s-dashboard-admin.yaml
```

```shell
kubectl describe secret -n kubernetes-dashboard k8s-dashboard-admin
```

```shell
kubectl proxy
```

[http://localhost:8001/api/v1/namespaces/kubernetes-dashboard/services/https:kubernetes-dashboard:/proxy/](http://localhost:8001/api/v1/namespaces/kubernetes-dashboard/services/https:kubernetes-dashboard:/proxy/)

```shell
kubectl describe secret -n kubernetes-dashboard k8s-dashboard-admin | grep token: | awk '{ print $2 }'
```
