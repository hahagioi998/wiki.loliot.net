## depends_on

Pulumi가 여러 리소스를 생성하거나 제거할 때, 자동으로 종속성을 추적합니다. 하지만, 어떤 상황에 따라 자동으로 추적하지 못할 수 있는데, 그런 경우 `depends_on` 옵션을 사용하여 명시적으로 종속성을 지정할 수 있습니다.

```python

# ...

_tags = {"Name": f"{common.cluster_name}-AmazonEKSClusterPolicy"}
amazon_eks_cluster_policy = aws.iam.RolePolicyAttachment(
    _tags["Name"],
    policy_arn="arn:aws:iam::aws:policy/AmazonEKSClusterPolicy",
    role=eks_role.name,
    opts=pulumi.ResourceOptions(parent=eks_role),
)

# ...

_tags = {"Name": f"{common.cluster_name}-eks-0"}
_tags.update(common.tags)
eks = aws.eks.Cluster(
    _tags["Name"],
    name=common.cluster_name,
    vpc_config=aws.eks.ClusterVpcConfigArgs(
        subnet_ids=subnet_ids,
    ),
    role_arn=eks_role.arn,
    version="1.21",
    tags=_tags,
    opts=pulumi.ResourceOptions(
        depends_on=[
            amazon_eks_cluster_policy,
            amazon_eksvpc_resource_controller,
        ]
    ),
)
```

## ignore_changes

`ignore_changes`에 속성을 추가하면 리소스를 업데이트 할 때 해당 속성이 변경되더라도 무시됩니다. `ignore_changes` 추가된 속성은 해당 리소스가 생성될 때에만 사용됩니다.

```python
# 이 scurity group은 cluster load balancer를 위한 것이기 때문에 ingress가
# 자동으로 수정될 수 있습니다.
elb_sg = aws.ec2.SecurityGroup(
    "elb-sg",
    vpc_id=vpc.id,
    tags={
        "kubernetes.io/cluster/hhk-cluster": "owned",
    },
    opts=pulumi.ResourceOptions(ignore_changes=["ingress"]),
)
```

## parent

리소스의 상위 항목을 지정할 수 있습니다.

```python {15}
_tags = {"Name": f"{cluster}-vpc"}
_tags.update(common_tags)
vpc = aws.ec2.Vpc(
    _tags["Name"],
    cidr_block="10.234.0.0/16",
    tags=_tags,
)

_tags = {"Name": f"{cluster}-igw"}
_tags.update(common_tags)
igw = aws.ec2.InternetGateway(
    _tags["Name"],
    vpc_id=vpc.id,
    tags=_tags,
    opts=pulumi.ResourceOptions(parent=vpc),
)
```
